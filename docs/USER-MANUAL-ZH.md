## 1. 認識 Synfig Studio

### 什麼是 Synfig Studio？

Synfig Studio 是一款免費且開源的 2D 動畫軟體，專為製作電影級品質的動畫而設計。它最大的特點是**自動補間（Tweening）**——你不需要逐幀繪製動畫，只需設定關鍵幀，Synfig 會自動計算中間的過渡畫面。

### 為什麼設計師該使用 Synfig？

| 優勢 | 說明 |
|------|------|
| **向量與點陣兼容** | 同時支援向量圖形和點陣圖素材 |
| **自動補間** | 大幅減少重複勞動，專注於創意表現 |
| **非破壞性編輯** | 圖層、濾鏡皆可隨時調整，不損壞原始素材 |
| **參數連結** | 將不同物件的屬性互相連結，實現聯動動畫 |
| **跨平台** | 支援 Windows、Linux、macOS |
| **完全免費** | GNU GPL v3.0 開源授權，無需任何費用 |

### 核心概念

- **圖層（Layer）**：Synfig 的一切建構在圖層之上。每個形狀、影像、特效都是一個圖層。
- **鴨子（Duck）**：畫布上的控制點，用來調整物件的位置、大小、切線等屬性。綠色代表中心點，青色代表半徑或控制點。
- **畫布（Canvas）**：工作區域，可以包含子畫布，形成層級結構。
- **BLine**：Synfig 對貝茲曲線路徑的稱呼，類似其他軟體中的「路徑（Path）」。

## 2. 安裝與啟動

### 下載與安裝

- **Windows / Linux / macOS 安裝包**：[安裝說明](https://synfig.readthedocs.io/en/latest/installation.html)
- **從原始碼建置**：[建置指南](https://synfig-docs-dev.readthedocs.io/en/latest/building/Building%20Synfig.html)

### macOS 使用 Homebrew 安裝

```bash
# 安裝相依套件
./1-setup-osx-brew.sh

# 建置專案
./2-build-osx-brew.sh
```

### 首次啟動

啟動後，Synfig 會顯示：
- **工具箱（Toolbox）**——位於左上方
- **可自訂的停靠面板（Dock Dialogs）**——位於下方和右側
- **畫布視窗（Canvas Window）**——主要工作區域

> **自動恢復**：Synfig 每 5 分鐘自動儲存一次，如果程式意外關閉，重新啟動時會自動恢復未儲存的工作。

## 3. 介面總覽

### 主要區域

```
+-------------------+---------------------------+
|                   |                           |
|    工具箱         |       畫布視窗             |
|    (Toolbox)      |       (Canvas)            |
|                   |                           |
```

### 停靠面板一覽

| 面板 | 功能 |
|------|------|
| **圖層面板（Layers）** | 管理、組織所有圖層，支援拖曳排序 |
| **參數面板（Params）** | 顯示選取圖層的所有可調整參數 |
| **工具選項（Tool Options）** | 當前工具的專屬設定 |
| **導航器（Navigator）** | 畫布縮圖預覽，快速平移和縮放 |
| **歷史記錄（History）** | 完整的復原／重做歷史 |
| **時間軸（Timetrack）** | 動畫時間軸，顯示各圖層的關鍵幀 |
| **關鍵幀面板（Keyframes）** | 管理動畫關鍵幀 |
| **曲線編輯器（Curves）** | 編輯動畫曲線，精細控制補間效果 |
| **音軌面板（Soundwave）** | 顯示音訊波形，同步音畫 |
| **畫布面板（Canvases）** | 管理多個畫布 |
| **子物件面板（Children）** | 層級結構導航 |

> **小技巧**：面板標籤可以拖曳到不同位置重新排列。如果不小心關閉了某個面板，到 **檔案 > 對話框** 可以重新開啟。

### 路徑工具

| 工具 | 說明 |
|------|------|

| 工具 | 說明 |
|------|------|

| **填充工具（Fill）** | 以前景色填充封閉區域 |
| **滴管工具（Eyedrop）** | 從畫布上擷取顏色 |
| **縮放檢視（Zoom）** | 放大或縮小畫布檢視 |
| **草圖工具（Sketch）** | 輔助草圖模式 |

### 使用貝茲曲線工具（BLine）繪製形狀

這是 Synfig 最重要的繪圖工具，掌握它是高效工作的關鍵：

1. 選取 **BLine 工具**
2. 在 **工具選項** 中勾選「填充（Fill）」和「描邊（Outline）」
3. 在畫布上點擊放置頂點
4. **拖曳**頂點可以建立切線，控制曲線弧度
5. 右鍵點擊頂點可以：
   - 刪除頂點
   - 分離切線（Split Tangents）
   - 封閉路徑（Loop）
6. 切換到其他工具或點擊「建立」按鈕完成繪製

完成後會建立兩個連結的圖層：
- **描邊圖層（Outline）**——路徑的邊框
- **區域圖層（Region）**——路徑的填充

> **調整線寬變化**：選取描邊圖層後按 `Alt+5` 顯示寬度控制點，拖曳控制點可以讓線條粗細產生變化。

### 圖層操作

#### 建立群組（封裝）

1. 在圖層面板中選取多個圖層（按住 `Ctrl` 點擊）
2. 右鍵 > **「封裝（Encapsulate）」**
3. 建立一個「內嵌畫布（Inline Canvas）」群組
4. 點擊箭頭可以展開或收合群組內容

#### 圖層順序

- 上方圖層會覆蓋下方圖層
- 濾鏡圖層會影響其**下方**的圖層
- 在群組內，濾鏡只影響同群組的圖層

### 混合模式

混合模式決定圖層之間如何合成，共 27 種：

| 常用混合模式 | 效果 |
|-------------|------|
| **合成（Composite）** | 標準透明度合成（預設） |
| **背後（Behind）** | 圖層渲染在下方圖層之後 |
| **在上方（Onto）** | 限制圖層效果在下方圖層的形狀範圍內 |
| **直接（Straight）** | 線性混合，忽略透明度 |
| **相加（Add）** | 亮度疊加 |
| **相乘（Multiply）** | 暗部加深 |
| **濾色（Screen）** | 亮部提亮 |
| **疊加（Overlay）** | 同時增強亮部和暗部對比 |
| **強光（Hard Light）** | 強烈的疊加效果 |
| **差異（Difference）** | 顯示兩圖層的差異 |
| **變亮（Brighten）** | 保留較亮的像素 |
| **變暗（Darken）** | 保留較暗的像素 |
| **色相（Hue）** | 僅取用上方圖層的色相 |
| **飽和度（Saturation）** | 僅取用上方圖層的飽和度 |
| **明度（Luminance）** | 僅取用上方圖層的明度 |
| **色彩（Color）** | 取用上方圖層的色度，保留下方明度 |

> **設計師常用技巧**：使用「在上方（Onto）」模式可以將效果限制在特定形狀內，類似 Photoshop 的剪裁遮罩（Clipping Mask）。

## 6. 色彩與漸層

### 前景色與背景色

工具箱下方有前景色/背景色選取器：
- 點擊前景色方塊開啟**色彩選擇器**
- 按 **R** 按鈕重置為黑白

### 上色方式

1. **建立時指定**：在工具選項中設定顏色，建立新形狀時自動套用
2. **填充工具**：選取填充工具，點擊畫布上的物件即可套用前景色
3. **參數面板**：選取圖層 > 在參數面板中雙擊「顏色」參數直接修改
4. **調色盤面板**：使用調色盤快速選色

### 漸層編輯

1. 使用**漸層工具**在畫布上拖曳建立漸層圖層
2. 切換到**選取工具**後，拖曳漸層的控制點調整方向和範圍
3. 在圖層的參數中雙擊「漸層」參數開啟**漸層編輯器**
4. 在漸層編輯器中：
   - 點擊色條下方新增色標
   - 拖曳色標調整位置
   - 雙擊色標修改顏色
   - 右鍵刪除色標

### 漸層類型

- **線性漸層**：從 A 點到 B 點的直線過渡
- **放射漸層**：從中心向外的圓形過渡
- **錐形漸層**：繞中心旋轉的顏色過渡
- **螺旋漸層**：螺旋形的顏色過渡
- **曲線漸層**：沿自訂路徑的顏色過渡

## 7. 動畫基礎

### 動畫原理

Synfig 採用**關鍵幀補間**動畫方式：
1. 在時間軸上設定**關鍵幀（Keyframe）**
2. 在不同關鍵幀修改物件屬性（位置、大小、顏色等）
3. Synfig 自動計算中間幀的過渡

### 啟用動畫模式

1. 點擊畫布右下角的**動畫模式按鈕**（或快捷鍵）
2. 按鈕會亮起，表示進入動畫錄製模式
3. 此時在時間軸上移動時間指針，對物件做的任何修改都會自動建立關鍵幀

> **重要**：完成動畫設定後記得**關閉**動畫模式，否則之後的編輯都會被記錄為動畫。

### 建立第一個動畫

**步驟 1：設定場景**
1. 確認動畫模式已關閉
2. 建立需要的圖形元素（圓形、矩形、路徑等）

**步驟 2：設定起始狀態**
1. 將時間指針移到第 0 幀
2. 開啟動畫模式
3. 調整物件到起始位置

**步驟 3：設定結束狀態**
1. 將時間指針移到目標幀（例如第 48 幀）
2. 移動物件到結束位置
3. Synfig 自動在兩個關鍵幀之間建立補間動畫

**步驟 4：預覽**
1. 按空白鍵播放預覽
2. 或使用 **預覽對話框** 生成即時預覽

### 補間模式

Synfig 提供多種補間計算方式：

| 補間模式 | 效果 | 適用場景 |
|----------|------|----------|
| **TCB** | 平滑的 Hermite 曲線 | 自然流暢的動態（預設） |
| **線性（Linear）** | 等速直線過渡 | 機械運動、勻速移動 |
| **常數（Constant）** | 瞬間跳變，無過渡 | 開關狀態、突然變化 |
| **停頓（Halt）** | 緩入或緩出效果 | 物體起步和停止 |
| **鉗制（Clamped）** | 受限的曲線補間 | 需要控制範圍的動畫 |
| **手動（Manual）** | 自訂控制點 | 完全自訂的運動曲線 |

### 曲線編輯器

在**曲線編輯器面板**中可以精細調整動畫曲線：
- X 軸代表時間
- Y 軸代表參數值
- 拖曳曲線上的控制點調整運動節奏
- 右鍵點擊控制點可以切換補間模式

## 8. 進階動畫技巧

### 參數連結（Linking）

參數連結是 Synfig 最強大的功能之一，它讓不同物件的屬性互相聯動。

#### 連結控制點

1. 選取兩個控制點（例如兩個圓的半徑控制點）
2. 右鍵 > **「連結（Link）」**
3. 修改其中一個，另一個會自動跟隨

#### 連結參數

1. 選取多個圖層
2. 在參數面板中右鍵點擊想連結的參數
3. 選擇 **「連結（Link）」**

> **實用範例**：將角色眼睛的位置連結到同一個控制點，移動控制點就能同時控制雙眼的視線方向。

### 值節點（Value Nodes）

Synfig 擁有超過 70 種值節點，提供強大的參數控制能力：

#### 數學運算節點
- 加、減、乘、除
- 平均值、加權平均
- 線性內插、範圍對映

#### 向量操作節點
- 向量分量提取（X、Y）
- 向量角度、長度計算

#### 曲線操作節點
- BLine 頂點計算
- BLine 切線計算
- BLine 寬度計算

#### 時間控制節點
- 時間迴圈（Time Loop）——讓動畫片段循環播放
- 定時交換（Timed Swap）——在特定時間切換值
- 頻閃（Stroboscope）——建立閃爍效果

### 時間控制圖層

| 圖層 | 效果 |
|------|------|
| **時間迴圈（Time Loop）** | 讓動畫片段無限循環 |
| **頻閃（Stroboscope）** | 控制幀率產生閃爍效果 |
| **自由時間（Free Time）** | 獨立的時間控制 |

### 切換圖層（Switch Layer）

切換圖層可以在不同子圖層之間切換顯示，適用於：
- 角色的不同表情
- 嘴型動畫（口型同步）
- 手勢切換

## 9. 特效與濾鏡

### 模糊效果

| 效果 | 說明 | 用途 |
|------|------|------|
| 高斯模糊（Blur） | 均勻柔化 | 景深效果、柔焦 |
| 放射模糊（Radial Blur） | 從中心向外的模糊 | 速度感、聚焦效果 |
| 動態模糊（Motion Blur） | 沿運動方向的模糊 | 運動物體的速度感 |

**套用模糊的步驟：**
1. 選取目標圖層
2. 右鍵 > **新增圖層 > 模糊** > 選擇模糊類型
3. 將模糊圖層放在目標圖層**上方**
4. 如需限制模糊範圍：
   - 選取模糊圖層和目標圖層
   - 右鍵 > 封裝
   - 將模糊圖層的混合模式改為 **「直接（Straight）」**

| 亮度鍵（Lumakey） | 根據亮度值去背 |
| 半色調（Halftone） | 模擬印刷網點效果 |

| 曲線彎曲（Curve Warp） | 沿曲線路徑彎曲 | 文字沿路徑排列、旗幟飄動 |
| 旋渦（Twirl） | 旋渦狀扭曲 | 水面、魔法效果 |
| 拉伸（Stretch） | 單軸拉伸 | 擠壓和拉伸動態 |
| 內外翻轉（Inside Out） | 反轉拓撲結構 | 特殊藝術效果 |

## 10. 骨骼動畫

### 什麼是骨骼動畫？

骨骼動畫讓你建立一套「骨架」來驅動角色的動作，不需要逐一移動每個部位。

### 建立骨架

1. 選取**骨骼工具（Bone Tool）**
2. 在畫布上點擊建立第一根骨骼（根骨骼）
3. 從現有骨骼末端拖曳建立子骨骼
4. 重複步驟建立完整骨架

### 綁定角色

1. 建立角色的圖形圖層（身體各部位）
2. 新增**骨架變形圖層（Skeleton Deformation）**
3. 將骨架與圖形圖層封裝在同一群組中
4. 調整骨骼的影響權重

### 骨骼動畫工作流程

1. 開啟動畫模式
2. 移動時間指針到目標幀
3. 旋轉或移動骨骼
4. Synfig 自動補間骨骼的中間姿態
5. 所有綁定的圖形會跟隨骨骼變形

> **設計師提示**：先用簡單的形狀測試骨架設定，確認動作正確後再替換為精細的角色圖形。

## 11. 匯出與輸出

### 原生格式

| 格式 | 說明 |
|------|------|
| **.sif** | Synfig 專案檔（XML 格式） |
| **.sifz** | 壓縮的 Synfig 專案檔（推薦） |

### 圖片匯出

| 格式 | 特點 |
|------|------|
| **PNG** | 支援透明度，品質無損 |
| **JPEG** | 檔案較小，不支援透明度 |
| **BMP** | Windows 點陣圖 |
| **GIF** | 支援動畫，色彩受限 256 色 |
| **OpenEXR** | 高動態範圍（HDR），專業後期用 |
| **PPM** | 跨平台點陣圖 |

### 影片匯出

| 格式 | 說明 |
|------|------|
| **FFmpeg** | 支援 MP4、AVI、MOV 等主流影片格式 |
| **GIF 動畫** | 適合網頁和社群媒體 |
| **PNG 序列幀** | 逐幀輸出，適合後期合成 |

### 其他匯出

| 格式 | 說明 |
|------|------|
| **SVG** | 可縮放向量圖，與其他設計軟體交換用 |
| **Sprite Sheet** | 精靈圖表，適合遊戲開發 |

### 匯出步驟

1. **檔案 > 匯出（Render）**
2. 設定輸出格式和路徑
3. 調整解析度和品質參數
4. 如果是影片，設定 FFmpeg 參數（編碼器、位元率等）
5. 點擊**匯出**開始渲染

## 12. 實戰工作流程

### 工作流程一：角色動畫

```
1. 規劃角色設計
   └── 草圖 > 分解身體部位 > 確定關節點

2. 繪製角色
   └── 使用 BLine 工具逐一繪製各部位
   └── 每個部位放在獨立圖層
   └── 用群組組織（頭、身體、左手、右手等）

3. 建立骨架
   └── 骨骼工具建立骨架
   └── 設定父子關係
   └── 綁定骨架到圖形

4. 製作動畫
   └── 開啟動畫模式
   └── 設定關鍵姿態
   └── 調整曲線編輯器的補間效果
   └── 加入次要動態（頭髮、衣物擺動）

5. 匯出成品
   └── 預覽確認
   └── 選擇匯出格式
   └── 渲染輸出
```

### 工作流程二：動態圖像（Motion Graphics）

```
1. 設計畫面構圖
   └── 背景、前景元素配置

2. 建立圖形元素
   └── 幾何形狀：圓形、矩形、星形
   └── 漸層和純色背景
   └── 文字標題

3. 設定動畫
   └── 元素進場動畫（位移 + 縮放 + 透明度）
   └── 利用參數連結同步多個元素
   └── 使用補間曲線打磨動態節奏

4. 加入特效
   └── 模糊創造景深
   └── 混合模式豐富視覺層次

5. 配合音訊
   └── 匯入音訊圖層
   └── 使用音軌面板同步音畫

6. 匯出
   └── FFmpeg 匯出 MP4
```

### 工作流程三：逐幀匯入動畫

```
1. 在其他軟體繪製逐幀畫面（如 Krita、Procreate）
2. 匯出為 PNG 序列幀
3. 在 Synfig 中建立「動畫檔案」值節點
4. 載入序列幀
5. 利用 Synfig 的圖層系統加入特效
6. 匯出最終影片
```

## 14. 常見問題

### Q：繪製的圓形看不見？
**A**：可能半徑為 0。選取圓形圖層，在參數面板中將半徑改為合適的值（例如 50）。

### Q：觸控板繪圖有問題？
**A**：到 **檔案 > 輸入設備** 中停用觸控板裝置。

### Q：動畫沒有效果？
**A**：確認已開啟**動畫模式**（畫布右下角的按鈕應為亮起狀態），且時間指針不在第 0 幀。

### Q：模糊效果沒有作用在正確的圖層上？
**A**：模糊圖層只影響**同群組中位於其下方**的圖層。將模糊圖層和目標圖層封裝到同一群組中，並確保模糊圖層在上方。

### Q：如何讓效果只作用在特定形狀內？
**A**：使用 **「在上方（Onto）」** 混合模式。將效果圖層和形狀圖層封裝在同一群組中，效果圖層的混合模式設為「Onto」。

### Q：匯出的影片品質不好？
**A**：在 FFmpeg 匯出設定中調高位元率，或先匯出為 PNG 序列幀後使用專業影片編輯軟體合成。

### Q：檔案太大怎麼辦？
**A**：使用 `.sifz` 格式（壓縮版）儲存。匯入的點陣圖建議先在外部軟體壓縮後再匯入。

### Q：如何匯入 SVG 檔案？
**A**：**檔案 > 匯入** 支援 SVG 格式。匯入後 SVG 中的路徑會轉換為 Synfig 的 BLine 圖層。

### Q：Synfig 意外關閉，檔案還在嗎？
**A**：Synfig 有自動恢復功能，每 5 分鐘自動儲存一次。重新啟動後會提示恢復未儲存的工作。
