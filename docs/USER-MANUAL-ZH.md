## 1. 認識 Synfig Studio

### 什麼是 Synfig Studio？

Synfig Studio 是一款免費且開源的 2D 動畫軟體，專為製作電影級品質的動畫而設計。它最大的特點是**自動補間（Tweening）**——你不需要逐幀繪製動畫，只需設定關鍵幀，Synfig 會自動計算中間的過渡畫面。

### 為什麼設計師該使用 Synfig？

| 優勢 | 說明 |
|------|------|
| **向量與點陣兼容** | 同時支援向量圖形和點陣圖素材 |
| **自動補間** | 大幅減少重複勞動，專注於創意表現 |
| **非破壞性編輯** | 圖層、濾鏡皆可隨時調整，不損壞原始素材 |
| **參數連結** | 將不同物件的屬性互相連結，實現聯動動畫 |
| **跨平台** | 支援 Windows、Linux、macOS |
| **完全免費** | GNU GPL v3.0 開源授權，無需任何費用 |

### 核心概念

- **圖層（Layer）**：Synfig 的一切建構在圖層之上。每個形狀、影像、特效都是一個圖層。
- **鴨子（Duck）**：畫布上的控制點，用來調整物件的位置、大小、切線等屬性。綠色代表中心點，青色代表半徑，橙色代表頂點，黃色代表切線。
- **畫布（Canvas）**：工作區域，可以包含子畫布，形成層級結構。
- **BLine**：Synfig 對貝茲曲線路徑的稱呼，類似其他軟體中的「路徑（Path）」。
- **元資料（Metadata）**：儲存在畫布中的隱形資訊，用於記錄格線大小、輔助線位置、洋蔥皮設定等畫布特定偏好。

## 2. 安裝與啟動

### 下載與安裝

- **Windows / Linux / macOS 安裝包**：[安裝說明](https://synfig.readthedocs.io/en/latest/installation.html)
- **從原始碼建置**：[建置指南](https://synfig-docs-dev.readthedocs.io/en/latest/building/Building%20Synfig.html)

### macOS 使用 Homebrew 安裝

```bash
# 安裝相依套件
./1-setup-osx-brew.sh

# 建置專案
./2-build-osx-brew.sh
```

### 首次啟動

啟動後，Synfig 會顯示：
- **工具箱（Toolbox）**——位於左上方
- **可自訂的停靠面板（Dock Dialogs）**——位於下方和右側
- **畫布視窗（Canvas Window）**——主要工作區域

> **自動恢復**：Synfig 每 5 分鐘自動儲存一次，如果程式意外關閉，重新啟動時會自動恢復未儲存的工作。

### 工作區管理 (Workspace Management)

Synfig 支援多種預設 UI 佈局，可以透過 **視窗 > 工作區** 切換：

- **預設 (Default)**：平衡的佈局，適合大多數繪圖與編輯任務。
- **合成 (Compositing)**：強化圖層與參數面板，適合進行特效合成與色彩調整。
- **動畫 (Animating)**：擴大時間軸與關鍵幀區域，方便精細打磨動畫細節。

> **自訂工作區**：你可以自由拖曳面板標籤，並透過 **視窗 > 工作區 > 儲存工作區** 來保存你最順手的佈局。

## 6. 色彩與漸層

### 前景色與背景色

工具箱下方有前景色/背景色選取器：
- 點擊前景色方塊開啟**色彩選擇器**
- 按 **R** 按鈕重置為黑白

### 上色方式

1. **建立時指定**：在工具選項中設定顏色，建立新形狀時自動套用
2. **填充工具**：選取填充工具，點擊畫布上的物件即可套用前景色
3. **參數面板**：選取圖層 > 在參數面板中雙擊「顏色」參數直接修改
4. **調色盤面板**：使用調色盤快速選色

### 漸層編輯

1. 使用**漸層工具**在畫布上拖曳建立漸層圖層
2. 切換到**選取工具**後，拖曳漸層的控制點調整方向 and 範圍
3. 在圖層的參數中雙擊「漸層」參數開啟**漸層編輯器**
4. 在漸層編輯器中：
   - 點擊色條下方新增色標
   - 拖曳色標調整位置
   - 雙擊色標修改顏色
   - 右鍵刪除色標

## 7. 動畫基礎

### 動畫原理

Synfig 採用**關鍵幀補間**動畫方式：
1. 在時間軸上設定**關鍵幀（Keyframe）**
2. 在不同關鍵幀修改物件屬性（位置、大小、顏色等）
3. Synfig 自動計算中間幀的過渡

### 建立第一個動畫

**步驟 1：設定場景**
1. 確認動畫模式已關閉
2. 建立需要的圖形元素（圓形、矩形、路徑等）

**步驟 2：設定起始狀態**
1. 將時間指針移到第 0 幀
2. 開啟動畫模式
3. 調整物件到起始位置

**步驟 3：設定結束狀態**
1. 將時間指針移到目標幀（例如第 48 幀）
2. 移動物件到結束位置
3. Synfig 自動在兩個關鍵幀之間建立補間動畫

## 8. 進階動畫與預覽技巧

### 洋蔥皮 (Onion Skin)

洋蔥皮能讓你同時看到當前幀、過去幀與未來幀的影像，方便對位與打磨動態。

- **開啟/關閉**：點擊工具列圖示或按 `Alt+O`。
- **設定過去與未來幀**：在畫布視窗上方的微調欄中調整。
- **關鍵幀模式**：可以設定只顯示關鍵幀處的洋蔥皮，減少視覺干擾。

### 低解析度預覽模式 (Low-Res Mode)

當動畫場景變得複雜時，即時預覽可能會變得卡頓。

- **快速切換**：按 `Ctrl+Grave` (鍵盤左上角波浪鍵) 開啟低解析度模式。
- **原理**：Synfig 會以較大的像素塊來渲染預覽，大幅提升渲染速度。
- **自訂**：你可以透過 `Ctrl+(` 和 `Ctrl+)` 來減少或增加像素塊的大小。

### 參數連結 (Linking)

參數連結是 Synfig 最強大的功能之一，它讓不同物件的屬性互相聯動。

#### 連結控制點

1. 選取兩個控制點（例如兩個圓的半徑控制點）
2. 右鍵 > **「連結（Link）」**
3. 修改其中一個，另一個會自動跟隨

## 10. 骨骼動畫

### 建立骨架

1. 選取**骨骼工具（Bone Tool）**
2. 在畫布上點擊建立第一根骨骼（根骨骼）
3. 從現有骨骼末端拖曳建立子骨骼
4. 重複步驟建立完整骨架

### 綁定與變形

- **骨骼設定模式**：關閉動畫模式時調整骨骼的原始長度與角度。
- **動畫錄製模式**：開啟動畫模式時旋轉骨骼，Synfig 會自動計算變形。
- **骨架變形圖層**：將此圖層加入角色群組中，並連結到骨架，即可驅動複雜的點陣圖或向量網格變形。

## 11. 匯出與進階功能

### 向量化器 (Vectorizer)

Synfig 內建強大的向量化工具，可以將匯入的點陣圖轉換為向量路徑。

1. 選取一個 **點陣圖 (Bitmap)** 圖層。
2. 右鍵選單選擇 **向量化 (Vectorize)**。
3. 在彈出的對話框中設定 **中線 (Centerline)** 參數（如閾值、精確度）。
4. 點擊確定後，Synfig 會自動生成對應的 BLine 圖層。

### Lottie 導出器

對於網頁動畫師來說，Lottie 是目前最主流的格式。

1. 完成動畫製作後，選擇 **檔案 > 匯出 > 選擇 Lottie 格式**。
2. Lottie 支援大多數基本的形狀、顏色與位移動畫。
3. 導出的 JSON 檔案可以直接用於 Web、iOS 與 Android。

## 12. 實戰工作流程

### 工作流程一：角色動畫 (骨骼驅動)

1. **繪製角色**：使用 BLine 工具繪製各部位並分層。
2. **建立骨架**：使用骨骼工具建立階層式骨架。
3. **綁定**：使用「骨架變形」圖層將骨骼連結至繪圖層。
4. **製作動畫**：開啟動畫模式，擺弄骨骼姿態。
5. **打磨**：使用曲線編輯器調整補間節奏。

## 14. 效能優化

當處理大型專案或高解析度動畫時，可以參考以下優化建議：

1. **調整執行緒**：在 **檔案 > 選項 > 系統** 中，根據你的 CPU 核心數調整「執行緒數量」，以加快算圖速度。
2. **活用低解析度模式**：在編輯動畫動態時，始終保持低解析度模式 (Ctrl+Grave)，這能讓你獲得更流暢的即時反饋。
3. **隱藏不必要的圖層**：在處理特定角色時，可以暫時關閉複雜背景或其他角色的可見性 (點擊圖層旁的眼睛圖示)。
4. **壓縮專案檔**：使用 `.sifz` 格式儲存，這能顯著減小檔案體積，加快讀取與寫入速度。

## 15. 常見問題

### Q：動畫沒有效果？
**A**：確認已開啟**動畫模式**（畫布右下角的紅燈應為亮起狀態），且時間指針不在第 0 幀。

### Q：如何匯入 SVG 檔案？
**A**：**檔案 > 匯入** 支援 SVG 格式。匯入後 SVG 中的路徑會自動轉換為 Synfig 的 BLine 圖層。

### Q：Synfig 意外關閉，檔案還在嗎？
**A**：Synfig 有自動恢復功能，每 5 分鐘自動儲存一次。重新啟動後會提示恢復未儲存的工作。
